# Body Sculpt Studio. Homepage Wireframe.
Body Sculpt Studio is a landing page design created for a Pilates and wellness studio. The goal of the project was to design a calm, elegant, and user-friendly website that reflects the mindful and balanced atmosphere of Pilates practice.

The layout focuses on clear content structure and intuitive navigation, allowing visitors to easily explore available classes, learn about the studio, and choose a suitable membership plan. Soft neutral colors, natural textures, and balanced typography were used to create a relaxing and welcoming visual identity.

Special attention was given to the user journey—from discovering the studio and class types to exploring pricing options and registering for classes. The design aims to provide a seamless and engaging experience while highlighting the studio’s services and community.
